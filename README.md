## 📖 创客理念
作为一名计算机专业的大学生，我理解的创客并非单纯的“动手爱好者”，而是以创新为核心、以实践为路径的创造者。创客精神在于不满足于现有知识的吸收，更强调将想法落地——从脑海中的创意，到用代码、工具、材料将其转化为具体的实物或应用。

在大学阶段，创客思维让我们跳出课本的框架，把课堂学到的编程、建模、焊接等技能融合起来，解决真实的小问题，比如设计校庆文创、制作智能小装置。创客也不是孤军奋战，而是乐于分享、协作，在尝试与试错中积累经验，哪怕是一次失败的3D打印、一段调试不通的代码，都是创客成长的一部分。这种“敢想敢造、知行合一”的态度，正是大学生创客最珍贵的特质。

## 🚀 创客创新项目
### 中期创客项目——基于开源3D打印的WiFi控制六轴机械臂
- 团队成员：刘宸宇（编程/调试）、束伟（3D打印/组装）
- 项目背景：设计并实现一款基于ESP32的WiFi控制六轴机械臂，通过3D打印获取结构件，搭配舵机实现六轴运动，提供网页端可视化控制界面。
- 核心步骤：开源项目检索与模型下载 → 3D打印所有结构件 → 嵌入式程序编写（舵机驱动、WiFi联网、网页界面） → 机械臂组装与接线 → 程序烧录与联调测试。
- 遇到的问题及解决：
  - 打印件尺寸偏差 → Bambu Studio微调缩放比例（+0.5%）重新打印；
  - 舵机角度抖动 → 优化servoMove函数，增加constrain限制，调整脉冲宽度（500-2400μs）；
  - WiFi连接不稳定 → 优化重连逻辑，缩短handleClient轮询间隔。
- 成果：通过浏览器访问ESP32的IP地址即可实时控制机械臂各关节（底座、大臂、小臂、手腕俯仰/旋转、夹爪），支持滑块调节、一键复位、抓取/松开，控制响应延迟＜200ms。
- 详情见：[中期创客项目](midterm-project.html)
